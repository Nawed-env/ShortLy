# ShortLy

# URL Shortener – Spring Boot + PostgreSQL (Neon)

A simple Bit.ly–style URL Shortener built using **Java + Spring Boot**.  
Users can shorten URLs, manage them, and track click statistics (IP, user-agent, referer).

---

## Features
- Create short URLs  
- Auto-generated short keys  
- Redirect to original URL  
- Store click analytics  
- View all saved links  
- PostgreSQL integration (Neon)  
- REST API based
